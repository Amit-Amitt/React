# 📘 React Learning Repository

This repository contains my React.js practice code created for learning purposes. It includes different examples to understand core React concepts step by step.

---

## 🚀 Purpose

- Learn and practice React basics  
- Understand components and JSX  
- Work with props and hooks  
- Explore routing and API integration  
- Build a strong foundation in React  

---

## ⚙️ Setup Instructions

### 1. Clone the repository
```bash
git clone https://github.com/Amit-Amitt/react.git
```

### 2. Navigate to project folder
```bash
cd react
```

### 3. Install dependencies
```bash
npm install
```

### 4. Run the project
```bash
npm run dev
```
